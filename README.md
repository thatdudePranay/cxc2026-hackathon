# Foresight

3D costmap and SLAM integration for visually impaired indoor navigation.

## Quick start (notebooks)

```bash
pip install -r requirements.txt
jupyter notebook notebooks/
```

**Notebooks:**
- `01_costmap_demo.ipynb` – 3D costmap visualization with Plotly (interactive), synthetic obstacles, height-layer slices
- `02_path_safety.ipynb` – Test `is_path_safe()` for ankle/torso/head heights
